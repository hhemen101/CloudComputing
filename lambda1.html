AWS LAMBDA PROVIDES:

  • Pay-per-use pricing - The monthly compute price is $0.0000166667 per GB-s and the monthly request price is $0.20 per million requests. The free tier provides 400,000 GB-s and 1 million requests per month.
  • Completely automated administration - Lambda manages all infrastructure which enables you to focus on building differentiated backend services.
  • Built-in fault tolerance - Lambda maintains compute capacity across multiple Availability Zones in each Region to help protect your code against machine or data center failures.
  • Flexibility - Lambda can be used with other AWS services like S3 to build a variety of apps.
  • Compatibility - Lambda can be integrated with over 200 AWS services!

THINGS TO CONSIDER:

  • Limited connectivity - Since Lambda functions are stateless, it may not be able to maintain a connection to a database or other sources.
  • Memory constraints - The function's memory allocation choice will dictate how much CPU and network bandwidth it can use, which may be a limitation for some applications.
  • Limited language support - Currently, Lambda only supports Java, Go, PowerShell, Node. js, C#, Python, and Ruby. If the application does not support one of these languages, then it will have to use a different service.
